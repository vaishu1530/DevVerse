Important C Language Interview Questions

Q1. What is C language?

A procedural programming language developed by Dennis Ritchie in 1972 used for system and application development.

Q2. What are the features of C language?

Fast, low-level memory access, structured language, portable, supports pointers and functions.

Q3. What is the difference between call by value and call by reference?

Call by value copies the value. Call by reference shares the address, so original value can be modified.

Q4. What is a pointer?

A variable that stores the memory address of another variable.

Q5. What is the difference between malloc() and calloc()?

malloc() allocates memory without initialization; calloc() allocates and initializes to zero.

Q6. What is a dangling pointer?

A pointer that refers to a memory location that has been freed or deleted.

Q7. What is recursion in C?

A function calling itself repeatedly until a base condition is met.

Q8. What is the purpose of the static keyword?

It preserves the value of a variable across function calls or restricts scope within a file.

Q9. What are storage classes in C?

auto, register, static, and extern — they define scope, visibility, and lifetime of variables.

Q10. Difference between structure and union?

In structures, each member has its own storage; in unions, all members share the same memory.

Q11. What is the use of sizeof() operator?

Returns the size (in bytes) of a data type or variable.

Q12. What is a segmentation fault?

An error caused by accessing memory that the program doesn’t own (e.g., dereferencing a NULL pointer).

Q13. What are header files?

Files with `.h` extension that contain function declarations and macro definitions.

Q14. Explain the use of `const` keyword.

Makes a variable value unchangeable after initialization.

Q15. What is command line argument?

Parameters passed to the main function via `int main(int argc, char *argv[])` from the command line.
