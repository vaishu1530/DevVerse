Enum in C

An enum (short for enumeration) in C is a user-defined type that consists of named integer constants. Enums help make a program more readable by using meaningful names instead of numbers.

Syntax

enum enum_name { constant1, constant2, constant3, ... };

By default, the value of the first constant is 0, the next is 1, and so on. You can also assign custom values.

Example: Weekdays Enum

#include <stdio.h> enum Weekday { MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY }; int main() { enum Weekday today; today = WEDNESDAY; if (today == WEDNESDAY) { printf("Mid-week day\\n"); } return 0; }

Output:
Mid-week day

Custom Values in Enum

enum Level { LOW = 1, MEDIUM = 5, HIGH = 10 };

Here, LOW is 1, MEDIUM is 5, and HIGH is 10.

Advantages of Using Enum

  • Makes the code more readable and maintainable.
  • Helps in grouping related constants together.
  • Reduces chances of assigning invalid values (compared to raw integers).

Tips

  • Use enums when variables can only take a limited set of values.
  • You can also use enums in switch statements.
  • Enums are usually stored as integers.
