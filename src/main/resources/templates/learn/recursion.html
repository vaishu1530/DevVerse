Recursion in C

Recursion is a programming technique where a function calls itself to solve a smaller part of the original problem. It's commonly used to solve problems that can be broken down into similar subproblems.

1. Syntax of Recursion

return_type function_name(parameters) { if (base_condition) return result; else return function_name(smaller_problem); }

2. Example: Factorial using Recursion

#include <stdio.h> int factorial(int n) { if (n == 0) return 1; else return n * factorial(n - 1); } int main() { int num = 5; printf("Factorial of %d is %d", num, factorial(num)); return 0; }

Output: Factorial of 5 is 120

How Recursion Works (Factorial Example):

factorial(5) → 5 × factorial(4)
→ 5 × 4 × factorial(3)
→ 5 × 4 × 3 × factorial(2)
→ 5 × 4 × 3 × 2 × factorial(1)
→ 5 × 4 × 3 × 2 × 1 × factorial(0)
→ 5 × 4 × 3 × 2 × 1 × 1 = 120

3. Points to Remember

  • Every recursive function must have a base case to stop recursion.
  • Without a base case, the function will call itself indefinitely (infinite recursion).
  • Recursive functions use stack memory, so too many recursive calls can cause a stack overflow.

4. Applications of Recursion

  • Factorial calculation
  • Fibonacci series
  • Tree/graph traversals
  • Tower of Hanoi
  • Backtracking problems (like Sudoku, Maze solving)
