Pointers in C

A pointer is a variable that stores the memory address of another variable. It is a powerful feature of the C language that allows for direct memory access, dynamic memory allocation, and efficient array manipulation.

Syntax

data_type *pointer_name;

Example: int *ptr; declares a pointer to an integer.

Basic Example

#include <stdio.h>

int main() {
    int num = 10;
    int *ptr = #  // Pointer stores address of num

    printf("Value of num: %d\n", num);
    printf("Address of num: %p\n", &num);
    printf("Pointer value (address): %p\n", ptr);
    printf("Value at pointer (dereferencing): %d\n", *ptr);
    
    return 0;
}

Explanation

  • & gives the address of a variable.
  • * is used to dereference a pointer (get the value stored at the address).
  • int *ptr declares a pointer to an integer.

More Info

  • Pointers are used for dynamic memory allocation with malloc(), calloc(), and free().
  • Pointers allow passing large structures/arrays to functions efficiently.
  • Pointer arithmetic can be used to iterate over arrays.
