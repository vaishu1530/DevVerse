Functions in C

A function in C is a block of code that performs a specific task. C programs can have predefined library functions (like printf()) and user-defined functions.

Why Use Functions?

  • To avoid repeating code (reusability).
  • To divide large programs into smaller, manageable parts.
  • To improve code readability and debugging.

Syntax

return_type function_name(parameters) { // body of function }

Example: Add Two Numbers

#include <stdio.h> int add(int a, int b) { return a + b; } int main() { int result = add(5, 3); printf("Sum = %d\\n", result); return 0; }

Output:
Sum = 8

Types of Functions

  • Library Functions: Already available (e.g., printf(), scanf()).
  • User-defined Functions: Created by the programmer (like add() above).

Function Declaration (Prototype)

Declaring a function before using it is called a function prototype. It tells the compiler about the function name, return type, and parameters.

int add(int, int);

More Info

  • You can return any data type from a function.
  • void return type is used when no value is returned.
  • Functions can call other functions.
