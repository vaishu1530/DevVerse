Control Statements in C

Control statements allow you to change the flow of execution in a C program. These include decision-making, looping, and jumping statements.

1. Conditional Statements

Syntax:

if (condition) { // code block } else { // code block }

Example:

int age = 18; if (age >= 18) { printf("You are eligible to vote."); } else { printf("You are not eligible."); }

2. Looping Statements

Types: for, while, do-while

// For loop example for (int i = 0; i < 5; i++) { printf("%d\n", i); }

3. Jumping Statements

Types: break, continue, goto

// Break example for (int i = 0; i < 5; i++) { if (i == 3) { break; } printf("%d\n", i); }
