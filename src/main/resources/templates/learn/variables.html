What are Variables?

Variables in C are used to store data values. Every variable must be declared with a data type before use.

Syntax

data_type variable_name;       // Declaration
variable_name = value;        // Assignment

Example

#include <stdio.h>

int main() {
    int age = 21;
    float height = 5.9;
    char grade = 'A';

    printf("Age: %d\\n", age);
    printf("Height: %.1f\\n", height);
    printf("Grade: %c\\n", grade);

    return 0;
}

Explanation

  • int: Stores whole numbers (e.g., 5, 10, -3)
  • float: Stores decimal numbers (e.g., 3.14, 5.9)
  • char: Stores single characters (e.g., 'A', 'z')
