What are Operators?

Operators are special symbols used to perform operations on variables and values in C. They are essential to writing expressions and controlling logic flow in a program.

Syntax

Example

#include <stdio.h>

int main() {
    int a = 10, b = 5;

    printf("Addition: %d\\n", a + b);
    printf("Subtraction: %d\\n", a - b);
    printf("Multiplication: %d\\n", a * b);
    printf("Division: %d\\n", a / b);
    printf("Modulus: %d\\n", a % b);

    return 0;
}

Explanation

  • +: Adds two numbers
  • -: Subtracts the second number from the first
  • *: Multiplies two numbers
  • /: Divides the first number by the second
  • %: Gives the remainder of division

Other operators like relational, logical, and bitwise are used in decision-making and complex expressions.
