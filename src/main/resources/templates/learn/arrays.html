Arrays in C

An array in C is a collection of variables of the same data type stored at contiguous memory locations. Arrays allow you to store multiple values in a single variable.

1. Syntax of an Array

data_type array_name[size];

Example:

int numbers[5]; // declaration of array with 5 elements

2. Initializing Arrays

You can initialize an array at the time of declaration.

int numbers[5] = {1, 2, 3, 4, 5};

3. Accessing Elements

Array elements are accessed using index starting from 0.

printf("%d", numbers[0]); // prints 1
Array Memory Representation:

Index → 0 1 2 3 4
Value → [1] [2] [3] [4] [5]

4. Looping Through Arrays

for(int i = 0; i < 5; i++) { printf("%d ", numbers[i]); }

5. Types of Arrays

  • One-Dimensional Arrays
  • Two-Dimensional Arrays (Matrices)
  • Multi-Dimensional Arrays

2D Array Example:

int matrix[2][3] = { {1, 2, 3}, {4, 5, 6} };
