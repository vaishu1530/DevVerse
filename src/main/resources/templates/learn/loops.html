Loops in C

Loops are used to execute a block of code repeatedly as long as a given condition is true. They help avoid code repetition.

Types of Loops:

  • for loop
  • while loop
  • do...while loop

1. for Loop Syntax

for(initialization; condition; increment) {
    // code to be executed
}

Example:

#include <stdio.h>

int main() {
    for(int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

2. while Loop Syntax

while(condition) {
    // code to be executed
}

Example:

int i = 1;
while(i <= 5) {
    printf("%d\n", i);
    i++;
}

3. do...while Loop Syntax

do {
    // code to be executed
} while(condition);

Example:

int i = 1;
do {
    printf("%d\n", i);
    i++;
} while(i <= 5);

More Info

  • for loop is used when the number of iterations is known.
  • while and do...while are used when condition-based repetition is needed.
  • do...while executes at least once even if the condition is false initially.
